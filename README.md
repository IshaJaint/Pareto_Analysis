# 📊 Dynamic Pareto Analysis Dashboard – Power BI

## 📌 Project Overview
This project presents an interactive Pareto Analysis Dashboard built in Power BI to analyze business performance using the 80/20 principle. The dashboard dynamically identifies top contributors across multiple dimensions and metrics.

The objective was to implement Pareto-related calculations using Power BI’s Visual Calculations layer while maintaining dynamic interactivity across all visuals and KPI cards.

---

## 🎯 Business Objective
To help decision-makers:
- Identify top-performing Products, Customers, or Countries
- Validate the 80/20 rule dynamically
- Analyze performance using Sales or Profit
- Adjust Pareto threshold interactively

---

## 🚀 Key Features

- 🔄 Dynamic Dimension Switching (Product | Customer | Country)
- 📈 Metric Selector (Sales / Profit)
- 🎚 Adjustable Pareto Threshold
- 🏆 Automated Top Contributor Classification
- 📊 Combo Chart (Bar + Cumulative Line)
- 📌 KPI Summary Cards:
  - Total Sales
  - Total Profit
  - Total Items in Selected Dimension
  - Top Contributors Count
  - Top Contributors %

---

## 🧠 Technical Implementation

- Pareto logic implemented using **Visual Calculations**
  - Rank
  - Running Total
  - Cumulative %
  - Category Classification
- Field Parameters used for dynamic dimension switching
- DAX measures used only for base aggregations
- Fully responsive filter context handling
- Optimized layout for analytical clarity
